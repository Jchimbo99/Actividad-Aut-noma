Introducción al Document Object Model (DOM)

Una visión general del DOM y sus aplicaciones en la programación web dinámica

¿Qué es el Document Object Model (DOM)?

El Document Object Model (DOM) es una interfaz de programación que permite manipular la estructura de un documento HTML o XML. A través de JavaScript, el DOM proporciona acceso a los elementos que conforman la página web, permitiendo modificar su contenido y estructura de manera dinámica.

El Objeto Document

El objeto document es el objeto básico en JavaScript para acceder a la página web. Ofrece propiedades y métodos para interactuar con los elementos que la componen. Ejemplos de propiedades importantes incluyen:

  • Document.body: devuelve el elemento body del documento.
  • Document.title: devuelve el título de la página.
  • Document.images: es un arreglo con las imágenes de la página.
  • Document.bgcolor: define o devuelve el color de fondo de la página.

Además, el objeto document utiliza métodos como open(), close(), y write() para interactuar con el documento.


        // Cambiar el título de la página
        document.title = "Nuevo Título";

        // Acceder al elemento body
        let bodyElement = document.body;
        console.log(bodyElement); // Muestra el 

        // Cambiar el color de fondo
        document.bgColor = "lightblue";

Recorrido por el Árbol del DOM

El DOM representa la estructura de la página como un árbol de nodos. Los nodos pueden ser de tipo Element o Text. Por ejemplo, el nodo <body> tiene hijos como <header>, <nav>, <footer>, entre otros.

Para recorrer el árbol, podemos utilizar las siguientes propiedades:

  • parentNode: devuelve el nodo padre.
  • childNodes: devuelve los nodos hijos.
  • firstChild: accede al primer nodo hijo.
  • lastChild: accede al último nodo hijo.
  • previousSibling: devuelve el nodo hermano previo.
  • nextSibling: devuelve el nodo hermano siguiente.

Modificación de Elementos en el DOM

Podemos modificar elementos HTML de diversas maneras. Por ejemplo:

  • innerHTML: permite cambiar el contenido HTML de un elemento.
  • setAttribute(atributo, valor): cambia el valor de un atributo.
  • Para modificar estilos: elemento.style.propiedad = estilo;

        // Cambiar el contenido de un elemento
        document.getElementById("demo").innerHTML = "Nuevo Contenido";

        // Cambiar el color de fondo de un elemento
        let elemento = document.getElementById("demo");
        elemento.style.backgroundColor = "yellow";

        // Cambiar un atributo
        let link = document.getElementById("myLink");
        link.setAttribute("href", "https://www.nuevo-enlace.com");

Adición de Manejadores de Eventos

Es posible añadir manejadores de eventos a los elementos mediante JavaScript. Por ejemplo:


        document.getElementById("miElemento").onclick = function() {
            alert("¡Has hecho clic!");
        };

Este código se ejecutará cuando el usuario haga clic en el elemento con el id miElemento.
