¿Qué es el Document Object Model (DOM)?

El DOM es una interfaz de programación de aplicaciones (API) que permite representar documentos HTML y XML como objetos. Facilita la manipulación dinámica del contenido de la página web, la estructura y los estilos mediante lenguajes como JavaScript.

Una página web es un documento. Este documento puede exhibirse en la ventana de un navegador o también como código fuente HTML. Pero en los dos casos, es el mismo documento. El modelo de objeto de documento (DOM) proporciona otras formas de presentar, guardar y manipular este mismo documento. El DOM es una representación completamente orientada al objeto de la página web y puede ser modificado con un lenguaje de script como JavaScript.

El W3C DOM estándar forma la base del funcionamiento del DOM en muchos navegadores modernos. Varios navegadores ofrecen extensiones más allá del estándar W3C, por lo que es importante tener cuidado al utilizarlas, ya que los documentos pueden ser consultados por navegadores que tienen DOMs diferentes.

El DOM permite a los desarrolladores crear páginas web interactivas y dinámicas. Sin el DOM, los desarrolladores tendrían que recargar una página completamente para reflejar cambios, lo que haría que las experiencias interactivas y en tiempo real no fueran posibles. Gracias al DOM, los cambios a una página web pueden ser gestionados en tiempo real, haciendo que la interacción con el usuario sea más fluida.

El DOM se basa en una estructura de árbol, donde cada nodo en el árbol representa una parte del documento, como un elemento HTML, un atributo o un texto. Estos nodos pueden ser accedidos, modificados o eliminados utilizando varios métodos disponibles en la API de JavaScript.

Además de la manipulación del contenido, el DOM también permite modificar la estructura de la página, agregar o eliminar elementos, y cambiar sus estilos. Los eventos, como los clics de los usuarios o la carga de la página, pueden ser manejados a través del DOM, lo que ofrece una manera poderosa de hacer que una página web reaccione a la interacción del usuario.

En resumen, el DOM transforma un documento HTML o XML en un objeto interactivo que puede ser manipulado programáticamente a través de JavaScript. Esta capacidad de interacción dinámica es lo que hace que el DOM sea esencial para el desarrollo web moderno.


// Accediendo a un elemento con id "header"
let header = document.getElementById('header');

// Modificando el contenido de ese elemento
header.innerHTML = 'Nuevo Título del Documento';

¿Cómo Funciona el DOM?

El DOM representa la estructura de un documento HTML o XML como un árbol jerárquico. Cada nodo de este árbol puede ser un elemento HTML, un atributo, texto, o incluso comentarios. Este modelo permite que cada nodo sea accesible y manipulable de manera independiente.

El navegador interpreta los documentos HTML y XML, y los convierte en una estructura en memoria que puede ser modificada en tiempo real. Esta estructura se representa como un árbol de nodos, donde cada etiqueta HTML se convierte en un nodo, y cada atributo o texto dentro de esa etiqueta se convierte en un subnodo. Por ejemplo, un documento HTML con un encabezado y un párrafo se representaría como un árbol con nodos para el encabezado y el párrafo.

Cada nodo en el DOM puede ser manipulado a través de JavaScript. Por ejemplo, los elementos HTML pueden ser seleccionados mediante sus identificadores o clases, luego se pueden modificar, agregar o eliminar elementos del DOM.

La estructura del DOM permite que se acceda y modifique cualquier parte del documento, desde etiquetas HTML hasta el texto dentro de un elemento. El navegador utiliza esta estructura para representar el documento y permitir su manipulación en tiempo real mediante JavaScript.

Esta representación jerárquica de los documentos permite una gran flexibilidad. Por ejemplo, se pueden agregar nuevos elementos, cambiar los estilos de los elementos existentes, o responder a eventos del usuario como clics o desplazamientos con un simple cambio en la estructura del DOM.

El DOM también facilita la creación de sitios web responsivos, que se adaptan a diferentes tamaños de pantalla y dispositivos. Con la manipulación del DOM, es posible crear interfaces de usuario dinámicas, donde los cambios en los datos o en la vista se reflejan sin necesidad de recargar la página.


// Accediendo a un nodo de un elemento HTML con clase "button"
let button = document.querySelector('.button');

// Cambiando el texto de un botón
button.innerHTML = 'Haz clic aquí';

Manipulación del DOM con JavaScript

JavaScript es el lenguaje de programación más utilizado para interactuar con el DOM. Mediante JavaScript, se pueden crear y manipular nodos del DOM, agregando elementos, cambiando atributos, o gestionando eventos.

El proceso de manipulación del DOM comienza con la selección de los nodos o elementos con los que deseas interactuar. Luego, puedes modificar sus propiedades, cambiar su estructura, o incluso agregar nuevos elementos al DOM.

JavaScript ofrece una serie de métodos y propiedades para manipular el DOM. Algunas de las funciones más comunes incluyen:

  • getElementById - Selecciona un nodo basado en su id. 
                    
document.getElementById('miElemento').innerHTML = "Nuevo contenido en el elemento";
  • getElementsByClassName - Selecciona nodos basados en su clase. 
                    
let elementos = document.getElementsByClassName('miClase');
for (let i = 0; i < elementos.length; i++) {
    elementos[i].style.color = 'red';
}
  • querySelector - Selecciona un solo nodo utilizando un selector CSS. 
                    
let primerElemento = document.querySelector('.miClase');
primerElemento.style.backgroundColor = 'yellow';
  • createElement - Crea un nuevo elemento HTML. 
                    
let nuevoParrafo = document.createElement('p');
nuevoParrafo.innerHTML = "Este es un nuevo párrafo creado con JavaScript";
document.body.appendChild(nuevoParrafo);
  • appendChild - Agrega un nuevo nodo como hijo de otro nodo. 
                    
let nuevoDiv = document.createElement('div');
nuevoDiv.innerHTML = '¡Este es un nuevo div!';
document.body.appendChild(nuevoDiv);
  • removeChild - Elimina un nodo hijo de un nodo especificado. 
                    
let elementoPadre = document.getElementById('miElementoPadre');
let elementoAEliminar = document.getElementById('miElemento');
elementoPadre.removeChild(elementoAEliminar);
